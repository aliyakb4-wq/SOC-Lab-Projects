# Commands Used

## Command 1

```bash
ping google.com
```

**Purpose:** Test network connectivity.

---

## Command 2

```bash
ping example.com
```

**Purpose:** Verify ICMP traffic.

---

## Command 3

```bash
nslookup google.com
```

**Purpose:** Generate DNS queries.

---

## Command 4

```bash
nslookup example.com
```

**Purpose:** Generate DNS queries.

---

## Command 5

```bash
wget https://example.com
```

**Purpose:** Generate HTTPS/TLS traffic.
